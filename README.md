🤖 LearnBot - Your Interactive Programming Companion
LearnBot is a desktop-based interactive learning platform designed to help beginners master fundamental and advanced programming concepts. Built with Python and Tkinter, it provides a structured learning path, real-world analogies, and a built-in assessment system to make coding education engaging and accessible.

🌟 Key Features
Comprehensive Knowledge Base: Covers a wide range of topics from basic variables and data types to complex algorithms and data structures.

Interactive Learning Interface: A clean, chat-like UI that provides detailed explanations and real-world analogies for every concept.

Structured Learning Paths: Topics are categorized by difficulty levels (Beginner, Intermediate, Advanced) to guide learners systematically.

Built-in Quiz System: Challenge yourself with interactive multiple-choice questions to test your understanding of the concepts learned.

Progress Tracking: Monitor your learning journey with a visual progress bar and performance summaries, including quiz scores and topics completed.

Search Functionality: Quickly find specific programming terms or concepts using the integrated search feature.
